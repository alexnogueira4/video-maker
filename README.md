# NODE Rest full API

### About

### Features
+ Restfull
+ MongoDB
+ Tests with Mocha
+ Express
+ Javascript
+ Passport
+ JSON Web Token - JWT


### Install
This project uses MongoDB as database. Make sure you have to installed MongoDB 
````
npm install --global mocha
npm install 
````

### Start
````
npm start
````

### Testing
The tests for each version of the API can be found in ````./test````, and each route has its own test file.

To run the tests:

````
npm test
````
